Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b>(não semantica)

Nesta frase, temos um termo em destaque usando a tag <strong>

Italico / Ênfase

Nesta frase, temos um termo em itálico usando a tag <I> (não semântica).

Nesta frase, temos um termo em ênfase usando a tag <em> (semânitca).

Podemos criar também um texto marcado usando a tag <mark>

E no outro parágrafo temos um outro texto marcado.

Mais um texto mark para poder mudar todos em css

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno neste parágrafo

Texto deletado

Podemos marcar um texto como excluído para indicar que ele nçao deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois.

Existe também o texto sublinhado com a tag <u> (não semânitca).

Texto sobrescrito

Para inserir coisas do tipo x20+3.
utilizando a tag <sup>

Texto subscrito

Para inserir coisas do tipo: H2O
utilizando a tag <sub>

Tags aprendidas nesta aula:

<strong>: Negrito no texto / Dar força ao trecho
<em>: Ênfase no texto, visualmente igual ao itálico.
<MARK> Para marcar o texto com caneta de marcação.
<small> texto pequeno.
<ins> sublinhar o texto. <del> marcar de maneira que esteja excluindo o texto.
